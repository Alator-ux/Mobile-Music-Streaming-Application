# 🎵 Music Streaming App (Internship Project)

> **Статус:** Учебный проект в рамках производственной практики

## 📖 О проекте
Данный репозиторий содержит исходный код мобильного приложения для потокового воспроизведения музыки. Проект разрабатывается в рамках **производственной практики** студентами ИММиКН ЮФУ.

Основная цель — исследование и реализация клиент-серверной архитектуры, работа с аудиопотоками, внедрение CI/CD и соблюдение стандартов безопасности при разработке мобильного ПО.
